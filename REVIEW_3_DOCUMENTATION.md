# Review 3 Documentation - LILNEST

## 1. Introduction
LILNEST is a maternal wellness platform built to support expectant mothers throughout pregnancy. The system brings together AI guidance, pregnancy-related tools, community support, and digital tracking into one application.

## 2. Project Description
The platform helps users manage wellness activities, access information, track progress, and use interactive features such as the pregnancy visualizer and time capsule. The aim is to reduce fragmentation by combining support, tracking, and guidance in one place.

## 5. Project Scope
The project covers:
- AI support for maternal queries
- Pregnancy growth visualization
- Personal wellness tools
- Community interaction
- Marketplace access for services
- Authentication and role-based access control

## 6. Technology Stack
- React and Vite for the frontend
- Tailwind CSS for styling
- Express for backend APIs
- Firebase Authentication for login and authorization
- Prisma and MySQL for data persistence
- Groq API for AI assistance

## 7. Review 3 Progress Status
The current implementation includes the main application structure, authentication flow, protected routes, wellness modules, and backend support routes. The project is functional and ready for refinement and final integration work.
